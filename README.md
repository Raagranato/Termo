##Como rodar o executavel
Para testar a aplicação baixe o .jar, nele o código esta compilado e é só executa-lo.

## Como compilar e rodar
**Compilando:**

No terminal, na pasta raiz do projeto (onde estão os arquivos `.java`), rode:
```
javac *.java
```

**Executando:**
No terminal, ainda na pasta raiz:
```
java Main
```
